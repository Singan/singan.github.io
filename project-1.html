Project Overview

JWT 토큰을 통한 인증 인가처리를 하여 스케일 아웃 시 네트워킹 비용을 줄일 수 있도록 하였습니다
채팅 기능을 위해 간단하게 사용할 수 있는 Stomp 를 통해 웹 소켓을 구현하였습니다. 현재 경매가 시작된 상품에 대한 처리를 위해 스프링 스케줄링을 통해 10분이 지난 입찰은 낙찰이 되도록 구현하였고 진행중인 입찰은 서버에서 HashMap 을 통해 담게 되었습니다. 만약 스케일 아웃을 하게 된다면 이런 부분이 문제가 될 것 같아 DB를 활용하여 담는 방법에 대한 연구가 필요할 것 같습니다.
